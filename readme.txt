== Description ==

This Plugin can be useful to showcase your organization's Services, Features,Pricing, and deployments more uniquely in websites. The Artoon Imagebox Block Plugin allows you to display images with clickable content.

Artoon Imagebox Block Plugin is customizable and user-friendly.

If you want to make custom gutenberg block for your website then it will become easy. Just send your requirements <a href="https://artoonsolutions.com/contact-us/">Here</a>.

== Installation ==

1. Upload the "Artoon Imagebox Block" file to plugin directory and Activate it.
